# Changelog

All notable changes to Quizora are documented here.

## [1.0.0] — 2026-06-01

### Features
- AI-powered quiz generation using Google Gemini 3.5 Flash
- Real-time multiplayer quiz battles via WebSocket
- 25+ pre-built quizzes across 5 categories (NEET/JEE, AI/ML, Computer Science, Science, General Knowledge)
- Creator tools: manual quiz builder with multi-question support
- Gamification: XP, levels, streaks, dynamic badge unlocks
- Global leaderboards with category filtering
- Personal scoreboard with analytics
- User profiles with avatar, bio, and stats
- Admin panel with metrics, moderation, and feature flags
- Meme generator for sharing quiz results
- Venture Startup Suite for business analytics
- PWA support: installable, offline-capable
- JWT authentication with session management
- Rate limiting and content moderation
- Dark mode with Neo-Brutalist design system
- Mobile-responsive with bottom navigation bar
- SEO optimized with Open Graph, Twitter Cards, structured data

### Technical
- React 19 + TailwindCSS v4 + Vite 6
- Express.js with WebSocket server
- TypeScript strict mode
- GitHub Actions CI/CD pipeline
- Comprehensive test suite
- Full documentation (Architecture, API, Deployment, Security)
